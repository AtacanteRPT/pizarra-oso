# pizarra oso 


-- instrucciones 
```
git clone https://github.com/AtacanteRPT/pizarra-oso.git
```

despues entra ala carpeta 

```
cd pizarra-oso
```
instalar las dependencias de nodejs

```
npm install
```

ahora ejecutar el servidor 

```
npm start
```
finalmente abrir un navegador en 

```
localhost:3000
```
